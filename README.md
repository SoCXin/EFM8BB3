## [描述](https://github.com/mcuyun/CA72/wiki) 

基于ARMv8-A架构，最早发布于2015年年初，在相同的移动设备电池寿命限制下，Cortex-A72能相较基于Cortex-A15的设备提供3.5倍的性能表现，相比于Cortex-A57也有约1.8倍的性能提升。
